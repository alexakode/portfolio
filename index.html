Om meg

Gjennom frontendkurset Kodehode har jeg bygget videre på kompetansen og lært å utvikle løsninger med HTML, CSS og JavaScript. Jeg er spesielt opptatt av tilgjengelighet, både språklig og visuelt, med farger som tilpasses etter brukere og system.

Med bakgrunn fra språkstudier og lokalisering har jeg utviklet en forståelse for struktur, tilpasning og lokalisering av fri programvare på flere platformer, blant andre Transifex og Launchpad.

For meg handler god frontendutvikling om å møte brukere der de er, enten det gjelder språk, skjerm eller preferanser.

Prosjekter

Demonstrasjon av fargesystem, implementasjon av Material-farger til forskjellige tema, med varianter for høy kontrast.

Demonstrasjon av responsiv meny med CSS Grid, bygget med Vite.
